Linux命令行下手动编译java工程

· 2022-10-24 ·

1. 编译无特定依赖的工程

# -d 输出目录
javac -d ./bin java/xyz/shxiaj/pso/*
# -cp 字节码PATH
java -cp ./bin: xyz.shxiaj.pso.Pso

2. 编译依赖jar

# 编译和运行都需要引入jar路径, -cp --classpath, 用法跟环境变量一样 用冒号: 分隔多个环境目录
javac -cp ./hutool-all-5.8.9.jar -d ./bin java/xyz/shxiaj/pso/*
java -cp bin:./hutool-all-5.8.9.jar xyz.shxiaj.pso.CmdTest
